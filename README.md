# Prelegal

> **Status: In Progress** — Expected completion: 2026-03-14

## Overview

Prelegal is a project that provides a curated dataset of legal document templates. The system will allow users to browse, select, and modify legal agreements for their needs.

## Project Status

This project is currently **in progress** and is expected to be completed within 1 week (by 2026-03-14).

### Planned deliverables

- `templates/` — A directory of markdown legal agreement templates sourced from [CommonPaper](https://github.com/CommonPaper), available under CC By 4.0
- `catalog.json` — A structured index of all templates with name, description, and filename
- License attribution file inside `templates/` for the CC By 4.0 content
